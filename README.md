## Overview
This project analyzes student academic performance data using Python, Pandas, and Matplotlib. 
The goal is to identify trends, patterns, and insights from a dataset containing information on 1,000 students.

## Objectives
* Analyze student performance data.
* Identify high-performing and low-performing students.
* Visualize score distributions.
* Generate meaningful insights from the dataset.

## Tools Used
* Python
* Pandas
* Matplotlib
* Jupyter Notebook
